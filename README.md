# parking-spot-control-api

Recursos:
- Sprig Boot
- Spring MVC
- Spring Data JPA
- Spring Validation
- Spring Security: Basic Auth - No Config, In Memory e With Database
- API com métodos: getAll, getOne, Post, Delete, Update, Pageable, e Global Custom Date.

Preparação Ambiente:
- JDK 17
- Maven
- Postman
- PgAdmin (Postgres)
- IDE (VSCode)
